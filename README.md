# job-portal-api
Backend REST API for a Job Portal application built using Spring Boot.

## Tech Stack
- Java 21
- Spring Boot
- Maven
- Spring Data JPA
- H2 Database (for development)

## Features (Planned)
- User registration
- Job posting
- Job search
- Apply for jobs
- Admin management

## Project Structure

src/main/java/com/sahil/job_portal_api

controller → API endpoints  
service → Business logic  
repository → Database operations  
entity → Database models  
dto → Data transfer objects

## Run the Project

Clone the repository:

git clone https://github.com/Sahilpawar-lab/job-portal-api

Run the application:

mvn spring-boot:run

Application runs on:

http://localhost:8080
